홈페이지 프로젝트를 진행하는데  
Git과 GitHub를 무조건 공부해야 한다는 걸 알고 있었다.  
<br/>
파일이 늘어나고  
구조가 복잡해지고  
언젠가는 협업도 해야 하니  
꼭 알아둘 필요가 있었다.  
혼자 작업해도  
버전 관리는 필수라고 생각했다.  
<br/>
그래서 프로젝트를 멈췄다기보다는,  
다음으로 가기 위해 잠시 돌아섰다.  
<br/>
Git과 GitHub는 써본 적이 없었다.  
특히 터미널에서 명령어를 직접 입력하는 방식이 어색했다.  
개념과 지금 나에게 필요한 기능만 빠르게 정리했다.  
<br/>
그리고 다시 프로젝트로 돌아왔다.
